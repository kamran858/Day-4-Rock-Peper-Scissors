Rock Paper Scissors Game
A Python implementation of the classic Rock Paper Scissors game where you play against the computer.
Description
Play the timeless game of Rock Paper Scissors against a computer opponent. The program uses ASCII art to display choices and determines the winner based on classic game rules.
How to Run

python task.py

How to Play
	1.	Run the program
	2.	Choose your option:
	•	Type 0 for Rock
	•	Type 1 for Paper
	•	Type 2 for Scissors
	3.	The computer randomly selects its choice
	4.	The winner is determined and displayed
Game Rules
	•	Rock beats Scissors
	•	Scissors beats Paper
	•	Paper beats Rock
	•	Same choice = Draw
